# Commands (Phase 3)

Slash commands that turn a skill's procedure into a one-step invocation.

- **/research** — kicks off the orchestrator-worker flow from the `multi-agent-orchestration` skill, spawning the worker subagent in `agents/`.
- **/autonomous-loop** — starts the lock-file autonomy loop from the `parallel-autonomous-agents` skill, with the lock and progress scripts in `scripts/`.
